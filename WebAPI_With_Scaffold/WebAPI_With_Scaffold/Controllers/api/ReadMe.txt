Wenn ein ASP.NET Core MVC Projekt auch zusätzlich eine WebAPI anbieten, befinden sich
Controller von WebAPI + MVC im selben Verzeichnis.

Um eine Trennung zu erreichen, wir im Verzeichnis Controller, zusätzlich ein Unterverzeichnis
mit dem Namen API angelegt und darin befinden sich alle Controller, die eine WebAPI Funktionalität anbieten.
